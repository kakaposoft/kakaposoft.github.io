動作環境

OS
Windows 10 / Windows 11（64bit）
CPU
x64対応プロセッサ（マルチコア推奨）
メモリ
8GB以上推奨（AI機能使用時は16GB以上推奨）
GPU
DirectX 11対応GPU（AI処理時はGPU搭載環境推奨）
ストレージ
500MB以上の空き容量（モデル・キャッシュにより増加する場合があります）
ネット接続
起動時のバージョン確認・更新確認のためインターネット接続が必要です
（動画・画像・AI処理データの送信は行いません）
ランタイム
Microsoft WebView2 Runtime（未インストールの場合は自動取得または手動インストール）

※AI機能の処理速度は、GPU性能・VRAM容量・モデルサイズにより大きく変動します。
※一部の環境（GPUドライバ、セキュリティソフト、特殊なコーデック環境等）では正常に動作しない場合があります。
