kakaposoft

Kakaposoftは、「あったらいいな」「ありそうでない」を形にする個人開発プロジェクトです。 日常で役立つ、高機能で使いやすいソフトウェアを開発・公開しています。

公開フリーソフト［Windows用］

Videosoft

あらゆる動画や画像を再生できる、シンプルで高機能なビューアーソフトです。

ファイルをドラッグ＆ドロップするだけで簡単に再生できます。 軽量で高速な動作を重視し、快適な視聴環境を提供します。

■対応形式

動画形式：MP4 / MKV / AVI / MOV / WMV など

画像形式：JPEG / PNG / BMP / WebP など

■こだわりポイントと技術的詳細

動画再生にはFFmpegを利用しており、幅広い形式への対応と安定した再生を実現しています。

画像表示では無駄のない描画処理を採用し、大容量画像でもスムーズに表示できます。

AI技術を活用した高画質化機能など、最新技術の導入にも取り組んでいます。
