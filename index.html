公開フリーソフト［Windows用］

Kakapo Vision Player [KVP]

動画・画像・音声を、ドラッグ＆ドロップですぐ再生。
シンプル操作と高機能を両立した Windows用フリーソフトです。

KVPページを見る（機能・ダウンロード）→ 動作環境 →
主な特徴
  • シンプルで扱いやすいビューア（無駄のない設計）
  • 動画・画像・音声をまとめて再生
  • 幅広いファイル形式に対応
  • AIによる高画質化などの拡張機能（実験的）
使い方（かんたん3ステップ）
  1. 見たいファイル（動画／画像／音楽）を用意
  2. KVPの画面にドラッグ＆ドロップ
  3. すぐ再生できます

※ うまく開けないときは「動作環境」を確認してください。

Kakapo Vision Player紹介動画

■対応形式

動画形式：MP4 / MKV / AVI / MOV / WMV など

画像形式：JPEG / PNG / BMP / WebP / HEIC / HEIF など

音楽形式：WAV / FLAC など

安心して使える理由

外部送信なし（※ライセンス認証のみ通信）
画像・動画・音声データは、すべてパソコン内で処理されます。
ファイル内容や画面情報が、外部サーバーへ送信されることはありません。

※ 通信が発生するのは、応援特典などのライセンス認証（証明）のときだけです。
※ 再生・表示・処理そのものはオフラインでも動きます（ネット不要）。

詳しくは プライバシーポリシー をご確認ください。

応援について（任意）

応援は任意です。
Kakapo Vision Player の基本機能（再生・表示など）は、無料でそのまま利用できます。
